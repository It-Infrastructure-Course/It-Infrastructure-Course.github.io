We can facilitate the autostarting of Wamp in the system tray by disabling UAC, which will eliminate the need to approve WAMP each time the system starts in order for the tray icon to be present in the system tray.

Search in the taskbar for UAC (user account controll settings)
The default level should be 2
We will change the level to 0, or to never notify us.
This is not recommened for production environments or deployment. However, during development it can make your life a lot easier! Please exercise discretion and use with caution!
