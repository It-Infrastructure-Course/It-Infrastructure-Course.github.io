Installing Ubuntu

Installing Ubuntu follows the same basic steps as above, but lacks the need for a license key. Further, during install you may select to perform a minimal install and not to install updates while installing Ubuntu, as this will slow it down considerably. I recommend that updates be completed at home as they extend the instllation time to approximately 30 minutes
